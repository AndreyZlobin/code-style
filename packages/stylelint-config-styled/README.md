# @self/stylelint-config-styled

Stylelint config for styled


# Usage

## Installation

### Npm
```shell
npm i stylelint @self/stylelint-config-styled --save-dev 
```

### Yarn
```shell
yarn add stylelint @self/stylelint-config-styled -D
```

```.stylelintrc```
```json
{
  "extends": ["@self/stylelint-config-styled"]
}
```
